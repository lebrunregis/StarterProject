Welcome!
This asset contains 2 versions of Liquid Volume. Please use the appropriate package according to the rendering pipeline of your project:

** Builtin Pipeline version **
 Import package from /BuiltIn folder.
 
** Universal Rendering Pipeline (URP) version **
 Import package from /URP folder.

Once imported, please read the instructions located in the Documentation folder of the imported asset and check demo scenes included.
